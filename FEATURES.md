# 📋 Node.js Lead Manager - Features Detail Document

Ye version un users ke liye hai jo advanced Node.js environment aur high-performance React frontend ka use karna chahte hain. Ye system cPanel/Plesk ke "Node.js Selector" par chalne ke liye optimize kiya gaya hai.

## 1. Advanced Dashboard
*   **Live Analytics**: Total leads, New leads, aur upcoming visits ka real-time data.
*   **Interactive Charts**: Recharts library ka use karke banaye gaye dynamic charts jo lead sources aur trends dikhate hain.
*   **Quick Actions**: Dashboard se hi naye leads add karne ya calls schedule karne ki suvidha.

## 2. Professional Lead Management
*   **Full CRUD**: Leads ko create, read, update, aur delete karne ka poora control.
*   **Smart Filtering**: Status, Source, aur Project ke hisaab se leads ko filter karein.
*   **Detailed Timeline**: Har lead ki poori history (calls, visits, remarks) ek hi jagah par.
*   **Lead Quality Tracking**: Leads ko 'Hot', 'Warm', ya 'Cold' mark karne ki suvidha.

## 3. Automation & Webhooks (Khaas Feature)
*   **Incoming Webhooks**: Facebook Ads, Google Forms, ya apni website se leads ko direct system mein lane ke liye webhook support.
*   **Field Mapping**: Bahar se aane wale data ko system ke fields ke saath map karne ki suvidha.

## 4. Communication Tools
*   **Call Logs**: Har call ki detail (duration, notes) save karein.
*   **Follow-up Reminders**: Aane wale follow-ups ke liye automatic reminders.
*   **WhatsApp Integration**: Ek click mein lead ko WhatsApp message bhejne ka option.

## 5. Multi-Project Support
*   Ek hi system mein multiple real estate projects (e.g., Signature, Shreemad) ko manage karne ki suvidha.

## 6. Technical Stack (Advanced)
*   **Frontend**: React 19 + Tailwind CSS (Modern & Fast).
*   **Backend**: Express.js (Robust & Scalable).
*   **Database**: SQLite (via better-sqlite3) - High performance file-based database.
*   **Bundled**: `esbuild` ka use karke poore backend ko ek single `server.cjs` file mein bundle kiya gaya hai taaki deployment aasaan ho.

## 7. Security
*   **JWT/Session Auth**: Secure login system.
*   **Role Based Access**: Admin aur Staff ke liye alag-alag permissions.
